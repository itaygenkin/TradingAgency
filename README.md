# AI Trading Agent

An automated AI agent built with Python and Gemini to analyze market data, and generates trading insights.

## Project Structure
- `src/`: Core logic including AI agent and data tools.
- `main.py`: The entry point to run the trading agent.
- `data/reports/`: Locally stored trading reports generated by the agent.

## Setup Instructions
1. Clone the repository:
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows).
4. Install dependencies: `pip install -r requirements.txt`
5. Create a `.env` file and add your `GOOGLE_API_KEY`.
6. Run the agent: `python main.py`
